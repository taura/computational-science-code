# 応用問題ロードマップ — 接続と到達目標

採用した応用13問: **A1 ガチャ, A4 パーコレーション, B1 2D熱伝導, B2 波動, B3 拡散, C1 N体, C2 Boids, D1 MNIST推論, D3 回帰の学習, D4 MLP学習, F1 PINN, G1 CG法, G2 べき乗法**。

これらが「これまでの練習問題 (01〜13)」とどうつながり, 相互にどう積み上がって, どんな**面白い到達目標**に届くかを示す。

## 2. 部品の地図 (既存 → 応用の流れ)

### ライン1: 行列積/行列ベクトル積 → AI
```
03 matvec ─┐
05 matmul ─┼─→ D1 MNIST推論 ─→ D3 回帰の学習 ─→ D4 MLP学習 ─→ F1 PINN
10 GPU matmul ─┤        (forward=行列積+活性化)  (勾配降下)   (forward+backprop)  (NN で PDE)
13 SIMD matmul ┘
```
- D1 は「**自分の行列積がそのまま AI 推論になる**」着地点。
- D3 は予測(matvec)+勾配(reduction)+反復, D4 はそれに backprop を足した学習, F1 はさらに自動微分で PDE を解く NN。

### ライン2: stencil (近傍平均) → 偏微分方程式
```
03 collapse(2) ─→ B1 2D熱伝導(定常/ヤコビ) ─→ B2 波動 ─→ B3 拡散
06 reduction(max 残差) ┘            └────────→ (時間発展へ拡張)
```
- B1 が stencil の主役 (collapse + 残差 reduction + GPU + 台数効果を一本で)。B2/B3 は B1 の機械を時間発展に拡張。

### ライン3: 反復解法 (matvec + 内積) → 線形代数・固有値
```
03 matvec ─┐
06 内積/縮約 ─┼─→ G1 CG法 (Ax=b を解く) ←─ B1 の係数行列(2Dラプラシアン)を解く
11 内積(SIMD)┘ └→ G2 べき乗法 (固有値・固有ベクトル)
                       ├→ ① 弦・膜の固有振動モード (同じラプラシアンの固有ベクトル = 定在波の形)
                       └→ ② すごろく/ランダムウォークの定常分布 (一番よく止まるマス)
```
- G1 と G2 は**同じ反復構造** (matvec → 内積/縮約 → ベクトル更新 → 収束判定)。一方が書ければ他方も近い。
- **B1 ↔ G1 ↔ G2 が同じ 2D ラプラシアン行列で貫ける**: 差分法 B1 で温度場を出し, CG (G1) で同じ方程式を解き, べき乗法 G2 で同じ行列の固有振動モードを出す = 「**同じ行列を, 解く/満たす/振動させる**」3つの道具。
- G2 の応用は2つ用意 (両方採用): **① 固有振動モード** (結果が定在波の「形」として出る。基本モードを出すには `σI − A` の単純なシフトが必要 — matvec は不変), **② 定常分布** (純粋なべき乗法。ルールを自前生成でき, 結果が「順位」として味わえる。PageRank の遊べる版)。

### ライン4: モンテカルロ / 確率
```
06 モンテカルロπ ─→ A1 ガチャ(期待値・分布) ─→ A4 パーコレーション(相転移)
04 schedule(dynamic) ───────────────────────┘  (試行ごとに手間が違う)
```
- A1 は reduction の意味ある拡張 (期待値・分散)。A4 は試行内に連結判定を持ち, 04 の負荷分散が効く。

### ライン5: O(N²) 相互作用 → N体・群れ
```
03 二重ループ ─┐
06 reduction ─┼─→ C1 N体直接法(重力) ─→ C2 Boids(群れの創発)
11–13 SIMD/ILP ┘  └→ SIMD/GPU で加速 (台数効果の見せ場)
```

## 3. 応用問題どうしの依存関係 (作る順番の制約)

```
A1 ガチャ          : 06(reduction)            … 単独で可
A4 パーコレーション : 04(schedule)+06          … 単独で可 (連結判定を含む)
B1 2D熱伝導        : 03(collapse)+06          … stencil の起点
  ├ B2 波動        : B1                        … B1 の拡張
  └ B3 拡散        : B1                        … B1 の拡張
G1 CG法            : 03(matvec)+06(内積)       … B1 の行列を解くと接続が映える
G2 べき乗法        : 03(matvec)+06            … G1 と双子。①固有振動モード ②定常分布 (両方)
C1 N体             : 03+06(+11–13,GPU)        … 単独で可
  └ C2 Boids       : C1 の構造                 … C1 の後が自然
D1 MNIST推論       : 03/05/13(行列積)          … AI ラインの起点
  └ D3 回帰        : D1 の部品+06(勾配)         … 学習の入口
      └ D4 MLP学習 : D3+backprop               … AI ラインの集大成
          └ F1 PINN: D4+自動微分+B系のPDE理解   … 最上級
```

推奨作成順 (前提を満たす順, 易→難):
**A1 → A4 → B1 → (B2, B3) → G1 → G2 → C1 → C2 → D1 → D3 → D4 → F1**

## 4. 面白い到達目標 (capstone) と, そこへの道

1. **「行列積が AI になる」**
   既習の行列積 (05/10/13) → **D1 で MNIST を認識** → **D4 で自分の NN を学習させ精度が上がる**。
   到達感: 「並列化した行列積が, そのまま手書き数字を当てる/学習する」。

2. **「同じラプラシアンを, 解く・満たす・振動させる」**
   **B1 差分法**で温度場 → **G1 CG法**で同じ方程式 (2Dラプラシアン) を解く → **G2 べき乗法**で同じ行列の**固有振動モード(定在波の形)**を出す → **F1 PINN**で NN に PDE を解かせる。
   到達感: 1つの行列を軸に, 差分・反復解法・固有値・機械学習の4つの流儀を体験。

3. **「相転移を目撃する」**
   **A4 パーコレーション**で p を動かすと臨界点で浸透確率が急変。
   到達感: 確率シミュレーションから物理の相転移が立ち上がる。

4. **「べき乗法で形と順位を出す」**
   **G2 べき乗法** → ① 弦・膜の**固有振動モード**(定在波の形。1Dは理論 sin と一致, 2Dは太鼓の模様) / ② すごろくの**定常分布**(一番よく止まるマス)。
   到達感: 1つの反復計算 (matvec の繰り返し) から, 振動の形や「止まりやすさ」の順位が立ち上がる。

5. **「宇宙と群れを動かす」**
   **C1 N体** (重力) / **C2 Boids** (群れ) を **SIMD+GPU** で加速。
   到達感: O(N²) の重い計算を本コースの全技法で実用速度に。

## 5. 「技法 × 題材」被覆マップ (どの問題でどの技法が主役か)

| 技法 | 主に使う応用問題 |
|---|---|
| parallel for / collapse | B1, B2, B3 (stencil), C1, 全行列系 |
| schedule (動的負荷分散) | A4 (パーコレーション) |
| reduction | A1, B1(残差), C1, D3, G1, G2 |
| GPU (target/teams/map) | B1, C1, D1, G1 (いずれも発展版で) |
| SIMD / ILP / ベクトル型 | C1, D1, G1 の高速化, (matmulカーネル) |
| 反復 (収束まで回す) | B1, G1, G2, D3, D4, F1 |

→ 採用13問で, 本コースの全技法が「意味のある計算」の文脈で再登場する設計。

## メモ
- 各問題は従来方式 (逐次/穴あきベースライン + 答えマーカー, C++/Fortran, ビルド確認) で作る。重いもの (D4, F1) はスキャフォルディングを厚めに。
- F1 PINN は自動微分が要るため最後。D4 が動いてから着手。
- G1 を作るときは行列に **B1 の 2D ラプラシアン** を使うと, B1↔G1 の接続が教材として光る。
